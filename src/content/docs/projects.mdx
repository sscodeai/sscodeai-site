## local-agent-toolkit

Agent の memory、skill 管理、実行 trace を local-first に扱う toolkit です。`agency-agents-ja` のような agent library や `superpowers-ja` のような skill pack を、必要な単位で配布・導入する周辺 tooling として追跡します。

- Repository: [`jnMetaCode/local-agent-toolkit`](https://github.com/jnMetaCode/local-agent-toolkit)
- Components: `engram` (memory), `skillet` (skill manager), `tracelet` (run tracing)
