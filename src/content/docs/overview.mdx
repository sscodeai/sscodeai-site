# SSCodeAI

SSCodeAI は、日本の IT 開発、SIer、受託開発、自社サービス、SaaS、EC、製造業 DX で使える AI coding workflow と agent library を整理するサイトです。

## Projects

### superpowers-ja

AI coding agent に計画、TDD、systematic debugging、code review、完了前検証、日本語の開発現場向け workflow を追加します。

- Claude Code / Cursor / Codex CLI / Gemini CLI / OpenCode などに対応
- 日本語 code review、障害報告、受入テスト仕様、Git 運用を補強
- `npx superpowers-ja` で project に install
- upstream / 中文版との違いは [superpowers 系 version の違い](/superpowers-ja/version-differences/) に整理

### agency-agents-ja

日本の現場で使いやすい AI 専門家 agent と workflow の library です。

- **266 agents** (⭐ 97 japan-original + 169 upstream-aligned) + **26 workflows**
- 要件定義、設計、QA、障害対応、PM、sales、support、legal、finance などを収録
- Backlog / Redmine / Jira、GitHub / GitLab / Bitbucket、kintone、LINE WORKS、Sansan、freee を想定
- 個人情報保護法 / インボイス制度 / 電帳法、製造業 DX、公共 sector の compliance を考慮
- 上流 [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents) (MIT) を fork し、日本市場向け原創 97 件 + 上流互換 169 件で構成
- `superpowers-ja` の `workflow-runner` と組み合わせて複数 role workflow を実行

## Site Architecture

この site は Astro + Starlight で構成します。

- Documentation: Starlight
- Blog and tutorials: MDX content
- Deployment: Cloudflare Pages
- Content sync: `scripts/sync-content.js`
