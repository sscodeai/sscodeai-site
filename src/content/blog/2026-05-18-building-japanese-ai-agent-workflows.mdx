# 日本の開発現場向け AI agent workflow を作る理由

AI coding agent は強力ですが、日本の IT 開発現場では「実装できる」だけでは足りません。

要件定義、受入条件、検収、障害報告、稟議、承認、監査証跡、個人情報、保守運用までつながって初めて、現場で使える workflow になります。

`superpowers-ja` は AI coding agent の作業手順を整え、`agency-agents-ja` は専門 role と成果物を整理します。

この blog では、両 project の設計、実務 scenario、tutorial、video transcript を公開していきます。
