## Public Information

Good candidates for this site:

- GitHub profile
- Public email or contact form
- Technical focus areas
- Open source projects
- Public case studies
- Languages and working regions

Avoid publishing private information such as home address, phone number, customer names without approval, contract details, or internal project data.
