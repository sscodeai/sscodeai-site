# Profile

SSCodeAI は、日本語で働く IT 開発 team のために AI coding workflow、agent library、実務 tutorial を整理する open source project です。

主な関心は、AI agent を「便利な補助」から「現場で説明できる作業手順」に近づけることです。要件、設計、実装、review、test、障害対応、運用引き継ぎまで、成果物と確認観点を日本の開発現場に合わせて整えます。

## Focus Areas

| Area | What SSCodeAI publishes |
| --- | --- |
| AI coding workflow | planning、TDD、debugging、code review、検証、release 前確認 |
| Japanese localization | 日本語の開発文化、検収、証跡、障害報告、運用引き継ぎに合わせた prompt / skill |
| Agent library | 要件定義、設計、QA、PM、support、finance、legal、sales などの専門 role |
| Documentation | 導入手順、tutorial、version differences、品質管理、release notes |
| Open source maintenance | upstream 追従、日本向け差分の明文化、generated docs の同期 |

## Projects

- [SSCodeAI site](/overview/) — 上記 project の docs、blog、tutorial、generated catalog をまとめる documentation site。

## Working Style

SSCodeAI の文書は、実務で共有しやすいことを重視します。

- 日本語で読みやすく、必要に応じて英語の technical term を併記する
- 生成物だけでなく、前提、判断理由、確認方法を残す
- upstream 由来の内容と日本向け original / adaptation を分けて記録する
- 個人情報、顧客情報、契約情報、内部仕様を公開文書に入れない
