# Release Notes

This page summarizes the public release history of `superpowers-ja`.

## v0.3.0

### Added

- `japanese-incident-report`
- `japanese-acceptance-test-spec`
- `japanese-code-review-graph`
- Japanese localization for `subagent-driven-development`
- Eval transcript workflow and initial scenario transcripts

### Focus

v0.3.0 strengthened Japanese IT delivery workflows around incident reporting, acceptance testing, graph-assisted review, and eval-based quality control.

## v0.1.0

### Initial Release

- Started as a Japanese IT development fork based on upstream `obra/superpowers`
- Added multi-tool install support
- Added Japanese bootstrap instructions
- Added initial Japanese original skills:
  - `japanese-code-review`
  - `japanese-commit-conventions`
  - `japanese-documentation`
  - `japanese-git-workflow`
